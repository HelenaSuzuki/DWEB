Hyperlinks

Definição

É uma referência dentro de um documento em hipertexto para outras partes do mesmo documento,
outros documentos, arquivos ou endereço eletrônico. Os links podem ser remotos, internos, locais,
com endereço de correio ou arquivos.

Marcação de Hiperlinks

A marcação de hiperlinks acontece utilizando a tag <a> e a propriedade "href" que faz a referência
para o conteúdo desejado.

Sintaxe: <a href="endereço">Conteúdo Referenciado Clicável</a>

Hiperlinks Externos ou Remotos

Sãp links que referenciam sites ou páginas da web. Na propriedade href utilizamos a URL
(Uniform Resource Locator) para a referência que desejamos.

Sintaxe: <a href="URL">Conteúdo Referenciado</a>

Hiperlinks Internos

São links que apontam a um lugar diferente dentro da mesma página e possibilita acessar
diferentes parágrafos ou seções quando o documento é extenso.

Sintaxe:

<a href="#referencia">Conteúdo Referenciado</a>
<a name="referencia"></a>

Hiperlinks Locais

São links que apontam para outras páginas dentro de um mesmo site.

Sintaxe: <a href="arquivo.html">Conteúdo Referenciado</a>

Hiperlinks em Imagens

É possível realizar hiperlinks imagens envolvendo a tag de imagem com a tag de hiperlink.

Sintaxe: <a href="referencia"> <img src="imagem.jpg"> </a>

Hiperlinks para downloads de arquivos

São links que levam para arquivos e possibilitam realizar downloads destes arquivos.

Sintaxe: <a href="arquivo.extensão" download="arquivo.extensão" type="tipo">Conteúdo</a>

Hyperlinks para endereço eletrônico

São links que apontam para um endereço eletrônico, email.

Sintaxe: <a href="mailto:email@email.com> email@email.com </a>

*Media Types

É de identificador padrão usado na Internet para indicar o tipo de dado que um arquivo contém. Link
de acesso: Media-Types

HTML

É uma Linguagem de Marcação de Hipertexto baseada em tags criada na década de 1990 por Tim
Berners-Lee.

A HTML é utilizada para publicação de conteúdo em hipertexto, tais como:
  • texto;
  • imagem;
  • áudio;
  • vídeo.

Estes elementos podem ser vinculados através de hyperlinks.

TAGS

As tags definem ao navegador a marcação de conteúdo.
As tags tem funcionalidades específicas. Sem o uso de tags o navegador exibe um texto plano.

As tags podem possuir atributos, ou seja, são parâmetros que definem algum tipo de informação extra ao navegador.
